# ⚡ Smart Household Energy Advisor (EnergyAdvisor Pro)

Welcome to **EnergyAdvisor Pro**, a smart, interactive dashboard designed to help households analyze their electricity consumption, predict future bills, and optimize appliance usage for maximum savings. 

Developed as a Hackathon Project, it focuses on delivering a visually premium and heavily data-driven experience with dynamic charts, AI-driven suggestions, and real-time simulators.

---

## 🚀 Features

- **📊 Advanced Data Visualization**: Dynamic, animated charts built with Chart.js to track your monthly consumption trend and analyze energy distribution across appliances.
- **🔮 AI Bill Prediction**: Calculates and predicts your next month's estimated electricity bill based on historical data.
- **⏱️ What-If Simulator**: Experiment with adjusting your daily usage runtime on high-consuming appliances and instantly see how much money you save per month.
- **💡 Smart Actionable Tips**: Get dynamically generated, appliance-specific tips to effectively cut down energy loss and run your home efficiently.
- **✨ Premium UI/UX Elements**: Beautifully crafted Neo-Glassmorphism UI, a slick initial booting screen, and interactive components designed for the modern web.

---

## 🛠️ Tech Stack
- **Frontend Core**: Vanilla HTML5, CSS3, JavaScript (ES6+).
- **Libraries**: [Chart.js](https://www.chartjs.org/) for beautiful, responsive canvas-based analytics.
- **Design Elements**: Neo-glassmorphism visuals, smooth transitions, custom SVGs.
- **Deployment**: Ready for static hosting (GitHub Pages, Vercel, Netlify).

## 💻 Getting Started

You do not need any complex backend environments to run this dashboard. It is fully client-side out of the box!

### Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/h4rsh740/Household-Energy-Advisor.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Household-Energy-Advisor
   ```
3. Open `index.html` in your favorite web browser or run it using Live Server (VS Code Extension).

### How to use
- Use the **Add Appliance Form** on the left to input your devices from the presets or add custom ones along with wattage and hours used per day.
- A **"Load Dummy Data"** toggle is provided for quick hackathon demonstrations to test the dashboard's analytics without manual data entry.
- Check the **What-If Simulator** to set up a target appliance and reduce usage hours to see exact potential savings.
